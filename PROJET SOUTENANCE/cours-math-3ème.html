Calcul Numérique

VIDÉO - CALCUL NUMÉRIQUE 3ÈME

Nombres relatifs

Opérations avec les nombres relatifs :

Addition : (+5) + (-3) = +2
Soustraction : (-4) - (+2) = -6
Multiplication : (-3) × (+4) = -12
Division : (-15) ÷ (+3) = -5

Règle des signes : même signe → positif, signes différents → négatif

Puissances

Définition : aⁿ = a × a × ... × a (n fois)

5³ = 5 × 5 × 5 = 125
10² = 10 × 10 = 100
2⁴ = 2 × 2 × 2 × 2 = 16
a⁰ = 1 (pour a ≠ 0)

Priorités des opérations

Ordre à respecter : Parenthèses → Puissances → Multiplication/Division → Addition/Soustraction

3 + 4 × 5 = 3 + 20 = 23
(3 + 4) × 5 = 7 × 5 = 35
2³ + 3 × 4 = 8 + 12 = 20

Notation scientifique

Un nombre en notation scientifique s'écrit a × 10ⁿ où 1 ≤ a < 10

4500 = 4,5 × 10³
0,0032 = 3,2 × 10⁻³
La vitesse de la lumière : 3 × 10⁸ m/s

💡 Propriétés des puissances

Propriété Formule Exemple
Produit aⁿ × aᵐ = aⁿ⁺ᵐ 2³ × 2² = 2⁵ = 32
Quotient aⁿ ÷ aᵐ = aⁿ⁻ᵐ 5⁴ ÷ 5² = 5² = 25
Puissance de puissance (aⁿ)ᵐ = aⁿ×ᵐ (3²)³ = 3⁶ = 729

Calcul Littéral

VIDÉO - CALCUL LITTÉRAL 3ÈME

Développement

Distributivité : k(a + b) = ka + kb

3(x + 4) = 3x + 12
-2(3x - 5) = -6x + 10
(x + 2)(x + 3) = x² + 3x + 2x + 6 = x² + 5x + 6

Factorisation

Mettre en facteur commun : ka + kb = k(a + b)

3x + 6 = 3(x + 2)
x² + 5x = x(x + 5)
4x² - 8x = 4x(x - 2)

Identités remarquables

À connaître par cœur :

(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
(a + b)(a - b) = a² - b²

Réduction d'expressions

Regrouper les termes semblables

3x + 2x² - x + 4x² = 2x + 6x²
5a - 2b + 3a + 4b = 8a + 2b
2(x + 3) - 3(x - 1) = 2x + 6 - 3x + 3 = -x + 9

💡 Applications des identités remarquables

Développement : (2x + 3)² = 4x² + 12x + 9
Factorisation : x² - 9 = (x + 3)(x - 3)
Calcul mental : 101² = (100 + 1)² = 10000 + 200 + 1 = 10201

Équations et Inéquations

VIDÉO - ÉQUATIONS 3ÈME

Équations du premier degré

Forme générale : ax + b = 0

2x + 3 = 7 → 2x = 4 → x = 2
3(x - 2) = 5 → 3x - 6 = 5 → 3x = 11 → x = 11/3
4x - 8 = 2x + 4 → 2x = 12 → x = 6

Inéquations du premier degré

Attention ! Quand on multiplie ou divise par un nombre négatif, on change le sens de l'inégalité.

3x + 2 < 8 → 3x < 6 → x < 2
-2x > 4 → x < -2 (on change le sens)
5 - 2x ≥ 1 → -2x ≥ -4 → x ≤ 2

Mise en équation de problèmes

Traduction d'un énoncé en langage mathématique

"Le double d'un nombre augmenté de 5 vaut 13" → 2x + 5 = 13
"La somme de deux nombres vaut 20" → x + y = 20
"Un nombre et son triple ont pour somme 24" → x + 3x = 24

Vérification des solutions

Toujours vérifier que la solution trouvée satisfait l'équation

Pour 2x + 3 = 7, solution x = 2
Vérif : 2×2 + 3 = 4 + 3 = 7 ✓

💡 Méthode de résolution

Étapes pour résoudre ax + b = c :

1. Simplifier chaque membre
2. Isoler les termes en x d'un côté
3. Réduire l'expression
4. Diviser par le coefficient de x
5. Vérifier la solution

Fonctions

VIDÉO - FONCTIONS 3ÈME

Fonctions linéaires

Forme : f(x) = ax

Exemple : f(x) = 3x
Représentation graphique : droite passant par l'origine
Proportionnalité : image proportionnelle à l'antécédent
Coefficient directeur : a = pente de la droite

Fonctions affines

Forme : f(x) = ax + b

Exemple : f(x) = 2x + 3
b = ordonnée à l'origine (valeur en x=0)
a = coefficient directeur
Représentation : droite ne passant pas par l'origine

Représentation graphique

Dans un repère orthonormé (O, I, J)

Pour tracer f(x) = 2x + 1 :
• Point A(0,1) car f(0)=1
• Point B(1,3) car f(1)=3
• Tracer la droite (AB)

Lecture graphique

Image → valeur de f(x) pour un x donné
Antécédent → valeur de x pour un f(x) donné
Exemple : si f(2)=5, alors image de 2 est 5 et antécédent de 5 est 2

Fonction carré

Introduction : f(x) = x²

Courbe : parabole
Symétrie : par rapport à l'axe des ordonnées
Minimum : en (0,0)
Tableau de valeurs : f(-2)=4, f(-1)=1, f(0)=0, f(1)=1, f(2)=4

Géométrie

VIDÉO - GÉOMÉTRIE 3ÈME

Théorème de Thalès

Configuration "papillon" ou "triangles emboîtés"

Si (BC) // (DE) alors :
AD/AB = AE/AC = DE/BC
Utilisation : calcul de longueurs inconnues
Réciproque : vérifier le parallélisme

Théorème de Pythagore

Dans un triangle rectangle :

AB² + AC² = BC²
BC étant l'hypoténuse
Exemple : triangle 3-4-5 → 3²+4²=9+16=25=5²
Réciproque : si AB²+AC²=BC² alors triangle rectangle

Trigonométrie

Dans un triangle rectangle :

sin(angle) = côté opposé / hypoténuse
cos(angle) = côté adjacent / hypoténuse
tan(angle) = côté opposé / côté adjacent
SOH-CAH-TOA : moyen mnémotechnique

Vecteurs

Translation → déplacement selon un vecteur
Coordonnées → (xᵦ - xₐ, yᵦ - yₐ)
Égalité → même direction, sens et longueur
Somme → relation de Chasles

Aires et volumes

Formules à connaître :

Sphère : V = (4/3)πR³, A = 4πR²
Cône : V = (1/3)πR²h
Pyramide : V = (1/3)×Aire base×h
Cylindre : V = πR²h, A latérale = 2πRh

Statistiques descriptives

Caractéristiques d'une série :

Médiane : valeur qui sépare la série en deux
Étendue : différence entre max et min
Quartiles : Q1, Q2 (médiane), Q3

Différents types de diagrammes :

Diagramme en bâtons
Histogramme (pour classes)

Probabilités

Définition et calcul :

Probabilité = nombre cas favorables / nombre cas possibles
0 ≤ P(A) ≤ 1
Événement certain : P=1
Événement impossible : P=0

Expériences aléatoires
